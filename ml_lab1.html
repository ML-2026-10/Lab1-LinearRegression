# Resolver duplicados parciales

def resolve_partial_dupes(group):
    """
    Resuelve los duplicados parciales para un grupo de filas con el mismo Patient ID. Si hay filas idnticas, se mantiene una sola. Si hay filas con diferentes valores de CVD Risk Score, se mantiene la fila con el valor de CVD que ms se repite entre sus duplicados, o, de lo contrario, el promedio de los valores de CVD entre sus duplicados.
    :param group: DataFrame con las filas que tienen el mismo Patient ID
    :return: Una sola fila representativa para el grupo de duplicados parciales
    """
    if len(group) == 1:
        return group.iloc[0]

    counts = group['CVD Risk Score'].value_counts()
    if counts.max() > 1:
        return group.loc[group['CVD Risk Score'] == counts.idxmax()].iloc[0]
    else:
        row = group.iloc[0].copy()
        row['CVD Risk Score'] = group['CVD Risk Score'].mean()
        return row

4. A partir del contexto y los datos compartidos, cmo representar la regresin lineal de forma matemtica? Indique el mtodo utilizado y el proceso para resolverlo.

Forma matemtica de la regresin lineal:

$$\hat{y} = \beta_0 + \sum_{j=1}^{p} \beta_j x_j$$

donde:

  • $\hat{y}$ es la prediccin del objetivo (riesgo cardiovascular o variable dependiente)
  • $\beta_0$ es el intercepto
  • $\beta_j$ son los coeficientes asociados a cada variable $x_j$
  • $p$ es el nmero de variables predictoras

En forma matricial:

$$\hat{y} = X\beta$$

donde $X$ es la matriz de diseo (observaciones $\times$ variables) y $\beta$ es el vector de coeficientes.

Mtodo utilizado: Lasso (Least Absolute Shrinkage and Selection Operator)

El mejor modelo usa regresin Lasso con hiperparmetro $\alpha = 0.1$, que resuelve el problema de optimizacin:

$$\min_{\beta} \left\{ \|y - X\beta\|_2^2 + \alpha \|\beta\|_1 \right\}$$

donde:

  • $\|y - X\beta\|_2^2 = \sum_{i=1}^{n}(y_i - \hat{y}_i)^2$ es la suma de errores cuadrados (trmino de ajuste)
  • $\alpha \|\beta\|_1 = \alpha \sum_{j=1}^{p}|\beta_j|$ es la penalizacin L1 (trmino de regularizacin)

Proceso de resolucin:

  1. Preprocesamiento:

    • Imputacin de valores faltantes (media para numricas, moda para categricas)
    • Estandarizacin de variables numricas: $x'_j = \frac{x_j - \mu_j}{\sigma_j}$
    • Codificacin one-hot para variables categricas

  2. Entrenamiento:

    • Se ajusta el modelo Lasso minimizando la funcin objetivo mediante algoritmos de descenso coordinado (coordinate descent) o LARS (Least Angle Regression).
    • La penalizacin L1 ($\alpha \|\beta\|_1$) tiende a llevar algunos coeficientes a exactamente cero, realizando seleccin automtica de variables.

  3. Validacin:

    • Se evala el modelo en el conjunto test usando MAE, RMSE y $R^2$.
    • El valor $\alpha=0.1$ fue seleccionado (al ojo).

Comparacin con OLS (Mnimos Cuadrados Ordinarios):

  • OLS resuelve: $\min_{\beta} \|y - X\beta\|_2^2$ Solucin cerrada: $\hat{\beta} = (X^\top X)^{-1}X^\top y$
  • Lasso agrega penalizacin L1, lo que mejora la generalizacin y reduce overfitting, especialmente con muchas variables.

5. En el ciclo de machine learning Qu tipos de sesgo podra afectar los resultados y por qu? Describe dos tipos de sesgo.

1. Sesgo de seleccin (Selection Bias)

Definicin: Ocurre cuando los datos de entrenamiento no son representativos de la poblacin objetivo donde se aplicar el modelo.

Cmo afecta este problema?

  • Si el dataset proviene solo de pacientes hospitalizados o de ciertos grupos demogrficos (p.ej., edad, regin, nivel socioeconmico), el modelo aprender patrones especficos de esa subpoblacin.
  • Al desplegar el modelo en la poblacin general, su desempeo se degradar porque las caractersticas y distribuciones son diferentes.
  • Ejemplo: Si el dataset sobrerepresenta pacientes de alto riesgo (diabticos, hipertensos), el modelo puede subestimar el riesgo en poblacin general sana o sobrestimar en ciertos grupos.

Consecuencias:

  • Mtricas optimistas en entrenamiento/validacin pero mal desempeo en produccin.
  • Decisiones clnicas errneas que pueden afectar negativamente a grupos subrepresentados.

2. Sesgo de medicin (Measurement Bias)

Definicin: Ocurre cuando las variables se miden con error sistemtico o con diferentes estndares entre grupos.

Cmo afecta este problema?

  • Si la presin arterial, colesterol o glucosa se miden con equipos diferentes, calibraciones incorrectas o protocolos inconsistentes, los valores registrados no reflejan la realidad clnica.
  • El modelo aprender relaciones espurias basadas en errores de medicin en lugar de relaciones causales reales.
  • Ejemplo: Si los pacientes diabticos son monitoreados con mayor frecuencia (ms mediciones de glucosa), sus registros pueden tener mayor variabilidad o valores ms actualizados que los no-diabticos, introduciendo sesgo en la variable "Fasting Blood Sugar".

Consecuencias:

  • Coeficientes sesgados que no reflejan la verdadera asociacin entre variables y riesgo cardiovascular.
  • Predicciones errneas para pacientes medidos con protocolos diferentes.
  • Inequidad: grupos medidos con menor precisin pueden recibir predicciones menos confiables.
